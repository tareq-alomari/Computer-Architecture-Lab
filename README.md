# معمارية الحاسوب - عملي

**Computer Architecture Lab — CS 221**  
**جامعة إبَّة — كلية علوم الحاسوب وتكنولوجيا المعلومات**

محتوى المقرر الدراسي لمادة **معمارية الحاسوب - الجانب العملي**، وتغطي أساسيات لغة التجميع MIPS باستخدام بيئة MARS.

| # | المحاضرة | الموضوع |

└── محاضرات معمارية الحاسوب - عملي/
    ├── المحاضرات/                           # 10 محاضرات
    │   ├── 01-مقدمة-إلى-MARS-وMIPS/
    │   │   ├── خطة المحاضرة 1.md            # خطة تدريسية مفصلة
    │   │   ├── خطة المحاضرة 1.pdf
    │   │   ├── تحليل المحاضرة 1.md          # تحليل المحتوى التعليمي
    │   │   ├── تحليل المحاضرة 1.pdf
    │   │   ├── ورقة عمل المحاضرة 1.md       # تمارين وأنشطة
    │   │   ├── lecture_01.asm               # أمثلة عملية MIPS
    │   │   └── *.pptx                       # عروض تقديمية
    │   ├── 02-العمليات-الحسابية-الأساسية/
    │   └── ... → 10-مناقشة-وتقييم-المشروع-النهائي/

كل محاضرة تحتوي على:
- **خطة المحاضرة** — أهداف، مدة زمنية، أنشطة، تقييم
- **تحليل المحاضرة** — تحليل المحتوى والجمهور المستهدف
- **ورقة عمل** — أسئلة نظرية، تمارين عملية، تحديات
- **كود MIPS** (`*.asm`) — أمثلة عملية قابلة للتشغيل
- **عرض تقديمي** — شرائح PowerPoint (للمحاضرة الأولى)

## 🚀 كيفية الاستخدام

### تشغيل أكواد MIPS
1. افتح MARS
2. `File → Open` → اختر ملف `.asm`
3. اضغط F3 للتجميع (Assemble)
4. اضغط F5 للتشغيل (Run)

### تحويل الخطط إلى PDF
```bash
python3 "محاضرات معمارية الحاسوب - عملي/أدوات/convert_to_pdf.py"
```
