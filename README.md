# معمارية الحاسوب - عملي

**Computer Architecture Lab — CS 221**  
**جامعة إبَّة — كلية علوم الحاسوب وتكنولوجيا المعلومات**

محتوى المقرر الدراسي لمادة معمارية الحاسوب - الجانب العملي (MIPS Assembly).

## هيكل المجلدات

```
معمارية حاسوب/
├── الشعار/                          # شعار جامعة إبَّة
├── توصيف مقرر معمارية الحاسوب - عملي.docx  # توصيف المقرر
├── محاضرات معمارية الحاسوب - عملي/
│   ├── المحاضرات/                   # المحاضرات (10 محاضرات)
│   │   ├── 01-مقدمة-إلى-MARS-وMIPS/
│   │   ├── 02-العمليات-الحسابية-الأساسية/
│   │   ├── ...
│   │   └── 10-مناقشة-وتقييم-المشروع-النهائي/
│   └── أدوات/                       # سكربتات وأدوات مساعدة
│       ├── convert_to_pdf.py
│       ├── style.css
│       └── دليل MIPS الشامل - شرح كل رمز.md
```

كل محاضرة تحتوي على:
- **خطة المحاضرة** — خطة تدريسية مفصلة (MD + PDF)
- **تحليل المحاضرة** — تحليل المحتوى التعليمي (MD + PDF)
- **ورقة عمل** — تمارين وأنشطة عملية
- **كود MIPS** — أمثلة عملية (`.asm`)
- **عرض تقديمي** (للمحاضرة الأولى)
