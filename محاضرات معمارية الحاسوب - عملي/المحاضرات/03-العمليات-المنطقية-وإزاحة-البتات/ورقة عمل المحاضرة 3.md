# ورقة عمل المحاضرة 3: العمليات المنطقية وإزاحة البتات

**المقرر:** معمارية الحاسوب - عملي | **الأسبوع:** الثالث

## 🎯 أهداف الورقة
- فهم تعليمات AND, OR, XOR المنطقية
- استخدام إزاحة البتات SLL, SRL
- تطبيقات معالجة البتات

## القسم الأول: أسئلة نظرية ومفاهيمية

### السؤال 1
أكمل جدول الحقيقة لكل عملية:

| A | B | A AND B | A OR B | A XOR B |
|---|---|---------|--------|---------|
| 0 | 0 | | | |
| 0 | 1 | | | |
| 1 | 0 | | | |
| 1 | 1 | | | |

### السؤال 2
ما تأثير التعليمات التالية على قيمة `$t0` إذا كانت `$t0 = 0x0000000F`؟

| التعليمية | قيمة `$t0` بعد التنفيذ |
|-----------|----------------------|
| `sll $t0, $t0, 4` | |
| `srl $t0, $t0, 2` | |
| `andi $t0, $t0, 0xFF00` | |
| `ori $t0, $t0, 0x00F0` | |

### السؤال 3
اشرح بمثال كيف يمكن استخدام تعليمة `andi` لفحص إذا كان عدد معين زوجيًا أم فرديًا.

## القسم الثاني: تمارين برمجية

### التمرين 1: فحص البتات
اكتب برنامج MIPS يقرأ عددًا صحيحًا من المستخدم ثم يفحص ويطبع إذا كانت البتة (bit) رقم 3 (الرابعة من اليمين) مُشغلة (1) أم لا.

**مثال:**
```
أدخل عدد: 8 (0b1000)
البتة رقم 3 مُشغلة ✅

أدخل عدد: 5 (0b0101)
البتة رقم 3 غير مُشغلة ❌
```

### التمرين 2: تشغيل وإطفاء البتات
اكتب برنامجًا يقرأ عددًا صحيحًا ثم:
1. يشغل (set) البتة رقم 5 باستخدام `ori`
2. يطفئ (clear) البتة رقم 2 باستخدام `andi`
3. يعكس (toggle) البتة رقم 0 باستخدام `xori`
اطبع الرقم بعد كل عملية.

### التمرين 3: تشفير بسيط
اكتب برنامجًا يقرأ حرفًا من المستخدم (كقيمة ASCII) ثم يقوم بتشفيره باستخدام عملية XOR مع مفتاح ثابت `0x55`، ثم طباعة الحرف المشفر وقيمته.

---

## القسم الثالث: سؤال التحدي

### التحدي 🏆
اكتب دالة (في برنامج MIPS كامل) تستقبل عددين صحيحين `a` و `b` وتقوم بتبادلهما **دون استخدام أي متغير مؤقت (temporary variable)**، فقط باستخدام عمليات XOR على المسجلات.

بعد التبادل، اطبع العددين للتأكد من صحة التبادل.

**مثال:**
```
قبل: a = 3, b = 7
بعد: a = 7, b = 3
```

**💡 تلميح:**
```
a = a XOR b
b = a XOR b
a = a XOR b
```
