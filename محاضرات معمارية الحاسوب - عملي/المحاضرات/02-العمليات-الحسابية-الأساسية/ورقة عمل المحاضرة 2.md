# ورقة عمل المحاضرة 2: العمليات الحسابية الأساسية

**المقرر:** معمارية الحاسوب - عملي | **الأسبوع:** الثاني

## 🎯 أهداف الورقة
- فهم تعليمات الجمع والطرح في MIPS
- استخدام `add` و `sub` و `addi`
- التعامل مع الأعداد الصحيحة بالإدخال والإخراج

---

## القسم الأول: أسئلة نظرية ومفاهيمية

### السؤال 1
ما الفرق بين التعليمات `add` و `addi`؟ اذكر مثالاً يوضح الفرق.

### السؤال 2
إذا كان `$t0 = 10` و `$t1 = 4`، ما قيمة `$t2` بعد تنفيذ كل تعليمة من التالي:

| التعليمية | قيمة `$t2` |
|-----------|-----------|
| `add $t2, $t0, $t1` | |
| `sub $t2, $t0, $t1` | |
| `addi $t2, $t0, -7` | |
| `sub $t2, $t1, $t0` | |

### السؤال 3
ماذا يحدث إذا تجاوز ناتج الجمع الحد المسموح به في تعليمة `add`؟ وما هي تعليمة `addu` وكيف تختلف؟

## القسم الثاني: تمارين برمجية

### التمرين 1: آلة حاسبة بسيطة
اكتب برنامج MIPS يقرأ ثلاثة أعداد صحيحة `a` و `b` و `c` من المستخدم ثم يقوم بحساب وطباعة نتيجة المعادلة:

```
result = a + b - c
```

### التمرين 2: تحويل درجات الحرارة
اكتب برنامجًا يحول درجة حرارة من Celsius إلى Fahrenheit باستخدام المعادلة:

```
F = (C × 9/5) + 32
```

**تلميح:** استخدم `addi` و `mul` (أو استخدم `add` المتكرر).

### التمرين 3: العمليات المركبة
اكتب برنامجًا يقرأ عددين صحيحين `x` و `y` ويحسب:

```
result = (2*x + 3*y) - (x - y)
```
اطبع النتيجة النهائية.

---

## القسم الثالث: سؤال التحدي

### التحدي 🏆
بدون استخدام تعليمات الضرب (`mul`) أو القسمة (`div`) المضمنة، اكتب برنامجًا يقوم بضرب عدد صحيح في 7 باستخدام تعليمات `add` و `sll` فقط.

**مثال:**
```
أدخل رقم: 5
5 × 7 = 35
```

**💡 تلميح:** `N × 7 = N×(4 + 2 + 1) = N×4 + N×2 + N×1`
