- MARS Simulator (ملف JAR) — موجود في `CA-Tools.rar`
- Logisim (لمحاضرات لاحقة) — موجود في `CA-Tools.rar`
- جهاز عرض
- سبورة للشرح
- اتصال إنترنت للتحميل

## تعليمات التثبيت (للمحاضر)

### 1. تثبيت Java
```bash
# التحقق من وجود Java
java -version

# إذا لم يكن مثبتاً:
# Windows: https://www.java.com/download/
# Linux:   sudo apt install default-jre
# macOS:   brew install --cask java
```

### 2. تحميل MARS
- الرابط الرسمي: https://courses.missouristate.edu/kenvollmar/mars/
- الملف: `Mars_4_5.jar`
- الحجم: ~2 MB

### 3. تشغيل MARS
```bash
# سطر الأوامر
java -jar Mars_4_5.jar

# أو: انقر نقراً مزدوجاً على الملف
```

### 4. التحقق من التثبيت
- يجب أن تظهر نافذة MARS الرئيسية
- شريط القوائم: File, Run, Tools, Help
- أربع نوافذ رئيسية: Edit, Registers, Memory, Output

## توزيع ملف MARS للطلاب
- حمّل الملف على Google Drive أو منصة الجامعة
- أو انسخه على USB
- الطريقة المثلى: يرسل رابط التحميل المباشر قبل المحاضرة بيوم

## نقاط التركيز
- الفرق بين `.data` و `.text`
- كيفية استخدام `li` لتحميل القيم
- ترتيب System Call: `$v0` ← رقم الخدمة، `$a0` ← المعامل
- إنهاء البرنامج بـ `li $v0, 10` ثم `syscall`
- **تأكد من تثبيت Java قبل MARS**
