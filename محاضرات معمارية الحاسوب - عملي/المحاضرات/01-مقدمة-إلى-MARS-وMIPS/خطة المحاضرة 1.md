## تعليمات التثبيت (خطوات مفصلة من دليل شرح العمل على MARS)

الدليل: `أدوات/شرح العمل على برنامج mars.pdf` (إعداد د. خالد القحصة)

### الخطوة 1: فك ضغط حزمة الأدوات CA-Tools.rar
- الملف موجود في `أدوات/CA-Tools.rar`
- استخدم WinRAR أو 7-Zip لفك الضغط
- محتويات الحزمة:
  - `jre-7u7-windows-i586.exe` — مشغل Java
  - `Mars4_3.jar` — برنامج MARS
  - `logisim-win-2.7.1.exe` — محاكي Logisim (لمحاضرات لاحقة)

### الخطوة 2: تثبيت Java Runtime
```bash
# التحقق من وجود Java
java -version

# إذا لم يكن مثبتاً:
# Windows: شغّل jre-7u7-windows-i586.exe من CA-Tools.rar
# Linux:   sudo apt install default-jre
# macOS:   brew install --cask java
```

**ملاحظة:** MARS يتطلب Java ليتمكن من العمل. بدون Java لن يعمل الملف.

### الخطوة 3: تشغيل MARS
```bash
# سطر الأوامر
java -jar Mars4_3.jar

# أو: انقر نقراً مزدوجاً على ملف Mars4_3.jar
```

### الخطوة 4: التعرف على واجهة MARS (من الدليل)

بعد تشغيل MARS، تظهر النوافذ التالية:

| النافذة | الوظيفة |
|---------|---------|
| **Edit** | محرر النصوص — لكتابة كود MIPS |
| **Registers** | عرض المسجلات (Name, Number, Value) + Coproc 0, Coproc 1 |
| **Text Segment** | عرض الكود المُجمّع: Bkpt, Address, Code, Basic, Source |
| **Data Segment** | عرض البيانات في الذاكرة: Address, Value (+x) بصيغة Hex و ASCII |
| **Run I/O** | نافذة الإدخال والإخراج — تظهر نتائج البرنامج |
| **MARS Messages** | رسائل الأخطاء والتشغيل |

### الخطوة 5: كتابة وتشغيل أول برنامج (وفقاً للدليل)

1. **File → Open** → اختر `mips001.asm` الموجود في مجلد المحاضرة
2. **Assemble (F3)** — لتجميع الكود والتحقق من الأخطاء
3. **Execute (F5)** — لتشغيل البرنامج
4. **Step (F7)** — للتنفيذ سطراً سطراً (مفيد لفهم تدفق البرنامج)
5. استخدم **Run I/O** لإدخال القيم ورؤية النتائج
6. **Go** — لتشغيل البرنامج حتى نقطة معينة

## نقاط التركيز
- الفرق بين `.data` و `.text`
- كيفية استخدام `li` لتحميل القيم
- ترتيب System Call: `$v0` ← رقم الخدمة، `$a0` ← المعامل
- إنهاء البرنامج بـ `li $v0, 10` ثم `syscall`
- **تأكد من تثبيت Java قبل MARS**
