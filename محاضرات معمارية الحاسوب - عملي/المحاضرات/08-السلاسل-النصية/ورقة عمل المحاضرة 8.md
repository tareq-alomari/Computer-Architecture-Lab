# ورقة عمل المحاضرة 8: السلاسل النصية

**المقرر:** معمارية الحاسوب - عملي | **الأسبوع:** الثامن

---

## 🎯 أهداف الورقة
- تمثيل السلاسل النصية باستخدام ASCII
- استخدام `lb` و `sb` للتعامل مع الأحرف
- معالجة السلاسل حرفًا حرفًا

---

## القسم الأول: أسئلة نظرية ومفاهيمية

### السؤال 1
كيف يتم تمثيل نهاية سلسلة نصية في MIPS؟ وما قيمة الـ Null terminator في نظام ASCII؟

### السؤال 2
إذا كانت السلسلة `"MIPS"` مخزنة في الذاكرة بدءًا من العنوان `0x10010000`، فأكمل جدول الذاكرة التالي:

| العنوان | القيمة (char) | القيمة (hex) |
|---------|--------------|-------------|
| 0x10010000 | 'M' | 0x4D |
| 0x10010001 | 'I' | |
| 0x10010002 | | |
| 0x10010003 | | |
| 0x10010004 | | |

### السؤال 3
ما الفرق بين `lb` (load byte) و `lw` (load word)؟ وما الحالات التي نستخدم فيها كل تعليمة عند التعامل مع السلاسل النصية؟

---

## القسم الثاني: تمارين برمجية

### التمرين 1: حساب طول السلسلة
اكتب برنامج MIPS يقرأ سلسلة نصية من المستخدم ويحسب طولها (عدد الأحرف) بدون استخدام `strlen` الجاهزة.

**مثال:**
```
أدخل سلسلة: MIPS
طول السلسلة: 4
```

**تلميح:** اقرأ الحرف الأول، وإذا لم يكن null، زد العداد وانتقل للحرف التالي.

### التمرين 2: تحويل الأحرف إلى كبيرة
اكتب برنامجًا يقرأ سلسلة نصية ويحول كل الأحرف الإنجليزية الصغيرة (lowercase) إلى أحرف كبيرة (uppercase)، ثم يطبع السلسلة المحولة.

**تلميح:** الفرق بين 'a' و 'A' في ASCII هو `0x20` (أو 32).

### التمرين 3: عكس سلسلة نصية
اكتب برنامجًا يقرأ سلسلة نصية ويطبعها معكوسة (مقلوبة).

**مثال:**
```
أدخل سلسلة: Computer
السلسلة المعكوسة: retupmoC
```

---

## القسم الثالث: سؤال التحدي

### التحدي 🏆
اكتب برنامجًا يتحقق مما إذا كانت سلسلة نصية هي **Palindrome** (تُقرأ من اليسار ومن اليمين بنفس الشكل). يجب أن يتجاهل البرنامج المسافات وعلامات الترقيم.

**مثال:**
```
أدخل سلسلة: racecar
السلامة متناظرة ✅

أدخل سلسلة: A man a plan a canal panama
السلامة متناظرة ✅

أدخل سلسلة: hello
السلامة ليست متناظرة ❌
```

**💡 تلميح:** استخدم مؤشرين - واحد في بداية السلسلة والآخر في نهايتها - وقارن الأحرف مع التحرك نحو المنتصف.
