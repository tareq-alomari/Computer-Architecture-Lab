اكتب برنامج MIPS يتضمن دالة `add_numbers` تستقبل عددين في `$a0` و `$a1` وتعيد الناتج في `$v0`، ثم قم باستدعائها من البرنامج الرئيسي.

**مثال:**
```
البرنامج الرئيسي:
    jal add_numbers
اطبع الناتج
```

### التمرين 2: دالة إيجاد العامل (Factorial)
اكتب دالة غير عودية لحساب مضروب عدد (Factorial) باستخدام حلقة.

```
factorial(n) = n × (n-1) × (n-2) × ... × 1
```

**مثال:**
```
أدخل n: 5
5! = 120
```

3. يجب حفظ المسجلات `$s0`-`$s7` (إذا استخدمتها) على المكدس في بداية الدالة واستعادتها في النهاية

## القسم الثالث: سؤال التحدي

### التحدي 🏆
اكتب دالة عودية (Recursive) لحساب **متتالية فيبوناتشي (Fibonacci)**. Fib(n) يُعرّف كالتالي:

```
Fib(0) = 0
Fib(1) = 1
Fib(n) = Fib(n-1) + Fib(n-2)
```

اطبع قيمة Fib(10) = 55.

**مثال:**
```
أدخل n: 10
Fib(10) = 55
```

**💡 تلميح:** 
1. تذكر حفظ `$ra` على المكدس في كل استدعاء عودي
2. الشرط الأساسي (base case): إذا كان n = 0 أو n = 1، أعد n مباشرة
3. في الحالة العودية: `fib(n-1)` + `fib(n-2)`
